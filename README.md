
# ⛪ 등대교회 재정관리 (Lighthouse Church Finance)

Electron + SQLite 기반의 교회 재정 관리 프로그램입니다.  
수입과 지출을 기록하고, 연도·월·주별 통계를 시각적으로 확인할 수 있습니다.

## 📦 설치 및 실행 방법

```bash
npm install
npm start
```

## 🧱 exe 설치파일 생성

```bash
npm run dist
```
빌드 후 `dist/` 폴더 내에 Windows 설치파일이 생성됩니다.

## ✨ 주요 기능
- 수입/지출 등록 및 관리
- 기간별 통계 그래프 표시
- PDF 및 CSV 내보내기
- 완전 오프라인 실행 (SQLite DB 저장)
